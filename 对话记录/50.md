## 阶段 16.2：时间步动画播放 v1

当前阶段目标：
- 在结果页增加时间步动画播放能力。
- 用户选择字段后，可以按时间步顺序自动刷新表面图。
- 本阶段做客户端内播放，不导出 GIF 或视频文件。

本阶段完成功能：
- 结果页新增 `播放动画` 按钮。
- 结果页新增 `暂停动画` 按钮。
- 结果页新增 `上一帧` 按钮。
- 结果页新增 `下一帧` 按钮。
- 结果页新增 `帧间隔` 设置，范围 `100 ms` 到 `5000 ms`，默认 `800 ms`。
- 播放动画时会按时间步下拉框顺序循环切换。
- 每一帧都会按当前字段和当前时间步重新请求 VTK Reader。
- 支持 `p` 压力字段动画。
- 支持 `U` 速度模长字段动画。
- 如果某一帧加载失败，动画会自动暂停。

本阶段页面与 UI 完成情况：

页面名 `ResultsTab`：
- 页面标题：`结果`。
- UI 组件：字段下拉框、时间步下拉框、帧间隔 `QSpinBox`、播放/暂停/上一帧/下一帧按钮。
- 按钮或菜单项：
  - `播放动画`：按时间步循环播放。
  - `暂停动画`：停止自动播放。
  - `上一帧`：切到上一个时间步并刷新图。
  - `下一帧`：切到下一个时间步并刷新图。
  - `加载所选字段表面图`：加载当前字段和当前时间步。
- 鼠标操作：
  - 用户先选择字段。
  - 用户可调整帧间隔。
  - 点击播放后自动刷新 3D 视图。
- 状态提示：
  - 播放时状态栏显示 `时间步动画播放中。`
  - 暂停时状态栏显示 `时间步动画已暂停。`
  - 播放时结果页显示当前字段、当前时间步和帧序号。

页面名 `VtkViewerDialog`：
- 页面标题：`FoamDesk 3D 视图`。
- UI 组件：matplotlib 3D 表面图。
- 显示逻辑：每次时间步变化都会重绘当前字段表面图。

本阶段完成设计：
- `MainWindow` 新增 `_visual_animation_timer`。
- 使用 Qt `QTimer` 实现动画帧推进。
- 新增 `_play_visualization_animation()`。
- 新增 `_stop_visualization_animation()`。
- 新增 `_advance_visualization_frame()`。
- 新增 `_step_visualization_frame(step)`。
- 将原来的 `_load_selected_field_surface()` 拆出 `_render_selected_field_surface()`，供单帧加载和动画播放复用。

本阶段数据流设计：
- 用户点击 `播放动画` -> 启动 `QTimer`。
- `QTimer.timeout` -> `_advance_visualization_frame()`。
- `_advance_visualization_frame()` -> 时间步下拉框切到下一项。
- `_render_selected_field_surface()` -> 按当前字段和当前时间步读取 VTK 数据。
- `VtkViewerDialog.plot_field_surface()` -> 重绘 3D 表面图。
- 用户点击 `暂停动画` -> 停止 `QTimer`。

本阶段核心业务场景完成进度：
- 可视化业务场景：完成时间步动画播放 v1。
- 字段选择业务场景：动画复用当前字段选择。
- 时间步业务场景：动画复用当前时间步列表。
- Case 隔离业务场景：动画读取当前 case。
