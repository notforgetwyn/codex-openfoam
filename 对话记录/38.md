## 阶段 15：压力表面云图 v1

当前阶段目标：
- 将之前的压力点云云图升级为更接近专业 CFD 后处理的压力表面云图。
- 从 VTK 读取 OpenFOAM case 的外表面面片，不再只显示离散点。
- 用 Python/matplotlib 在独立 3D 视图窗口中按压力 `p` 给表面面片着色。
- 本阶段不做字段选择器、时间步选择器、切面云图和速度流线。

本阶段完成功能：
- 结果页新增 `加载压力表面云图` 按钮。
- 点击后读取当前项目的真实 OpenFOAM case。
- 通过 VTK 获取 `vtkCompositeDataGeometryFilter` 输出的外表面 `poly_data`。
- 从 `poly_data.GetPolys()` 提取表面面片连接关系。
- 从 `poly_data.GetPoints()` 提取表面点坐标。
- 从点字段 `p` 提取压力值。
- 对每个表面面片计算平均压力值。
- 使用 `Poly3DCollection` 绘制连续表面面片。
- 使用 `turbo` 色图按压力值给每个面片着色。
- 显示压力颜色条 `p`。
- 对大网格面片数量进行采样限制，避免 matplotlib 3D 卡死。

本阶段完成设计：
- `VtkViewerDialog` 新增 `plot_pressure_surface()`，负责表面面片渲染。
- 新增 `_faces()`，从 VTK polydata 的 polygon 数据中解析面片点索引。
- 新增 `_sample_faces()`，限制大网格下最多绘制的面片数量。
- 新增 `_scalar_mappable()`，用于构造 matplotlib 颜色条。
- 主窗口新增 `_load_pressure_surface_cloud()`，负责读取当前项目、调用 VTK 服务、校验 `p` 字段、触发表面渲染。
- 保留 VTK 作为 OpenFOAM 数据读取层，保留 matplotlib 作为 WSL 下稳定 GUI 渲染层。

本阶段数据流设计：
- 用户点击 `加载压力表面云图` -> `MainWindow._load_pressure_surface_cloud()`。
- `_load_pressure_surface_cloud()` -> `OpenFoamVtkService.inspect()` -> 获取字段列表和时间步。
- `_load_pressure_surface_cloud()` -> `OpenFoamVtkService.build_geometry_filter()` -> 获取外表面 `poly_data`。
- `poly_data.GetPoints()` -> numpy 点坐标。
- `poly_data.GetPolys()` -> 面片连接关系。
- `poly_data.GetPointData().GetArray("p")` -> numpy 压力数组。
- 每个面片 -> 根据面片顶点索引取压力 -> 计算平均压力 -> 映射到颜色。
- `Poly3DCollection` -> matplotlib 3D 窗口渲染表面云图。
