## 阶段 14：项目优先启动页与上次项目记忆

当前阶段目标：
- 调整启动流程：用户第一次打开软件时，先看到类似 PyCharm 的项目选择页，而不是直接进入主工作台。
- 用户选择、新建或打开项目后，再进入 FoamDesk 主工作台。
- 如果用户之前已经选择过项目，并且项目目录仍然有效，则下次启动直接进入主工作台并恢复该项目。
- 本阶段不改变 OpenFOAM 求解、结果可视化和参数配置能力，只调整启动入口和项目记忆。

本阶段完成功能：
- 新增 `FoamDesk - 选择项目` 启动页。
- 启动页展示当前工作区下已有项目列表。
- 支持双击已有项目进入主工作台。
- 支持在启动页点击 `新建项目`，创建项目后直接进入主工作台。
- 支持在启动页点击 `打开项目`，选择已有 FoamDesk 项目目录后进入主工作台。
- 支持在启动页点击 `刷新`，重新加载工作区项目列表。
- 新增上次项目记忆：打开、创建或切换项目后自动保存 `last_project_path`。
- 下次启动时，如果 `last_project_path` 指向有效项目，则跳过启动页，直接进入主工作台。
- 如果上次项目不存在、损坏或缺少 `project.json`，则回退到项目选择启动页。

本阶段页面与 UI 完成情况：

页面名 `StartupWindow`：
- 页面标题：`FoamDesk - 选择项目`。
- UI 组件：`QDialog`、`QLabel`、`QListWidget`、`QPushButton`。
- 按钮或菜单项：
  - `新建项目`：弹出项目名称输入框，创建成功后进入主工作台。
  - `打开项目`：打开目录选择器，选择有效 FoamDesk 项目后进入主工作台。
  - `刷新`：重新读取工作区下的项目列表。
- 键盘操作：项目列表支持 Qt 默认激活行为。
- 鼠标操作：
  - 双击项目列表项进入主工作台。
  - 点击 `新建项目` 创建新项目。
  - 点击 `打开项目` 打开本地项目。
  - 点击 `刷新` 更新项目列表。
- 状态提示：启动页底部提示 `双击项目进入主工作台。FoamDesk 会记住最后一次打开的项目。`

页面名 `MainWindow`：
- 页面标题：`FoamDesk`。
- UI 组件：原有主工作台组件不变。
- 按钮或菜单项：原有 `新建项目`、`打开`、项目树点击逻辑保留。
- 键盘操作：未新增。
- 鼠标操作：在主工作台内创建、打开或点击项目树切换项目时，都会更新上次项目记忆。
- 状态提示：
  - 从上次项目恢复时显示 `已恢复上次项目。`
  - 创建项目后显示 `项目创建完成。`
  - 打开项目后显示 `项目打开完成。`
  - 点击项目树切换项目后显示 `项目已切换。`

本阶段完成设计：
- `AppSettings` 新增 `last_project_path` 字段，用于记录最后一次打开的项目路径。
- `AppSettingsService` 读写 `config/settings.yaml` 时支持 `last_project_path`。
- `ProjectService` 新增 `remember_project(project)`，统一负责保存上次项目。
- `ProjectService` 新增 `open_last_project()`，统一负责尝试恢复上次项目，并在无效时安全返回 `None`。
- `MainWindow` 支持 `initial_project` 参数，主窗口创建后自动激活初始项目。
- `StartupWindow` 只负责启动选择，不承担主工作台业务。
- 启动流程从“直接创建 MainWindow”改为“先恢复上次项目；失败则显示 StartupWindow；选择项目后再创建 MainWindow”。

本阶段数据流设计：
- 软件启动 -> `ApplicationContext` 初始化服务 -> `ProjectService.open_last_project()`。
- 如果上次项目有效 -> `MainWindow(context, initial_project=project)` -> `_activate_project()` -> 加载项目、参数、求解页、结果页。
- 如果上次项目无效或不存在 -> 显示 `StartupWindow` -> 用户双击项目 / 新建项目 / 打开项目 -> `ProjectService.remember_project()` -> 返回 selected project -> 创建 `MainWindow`。
- 主工作台创建项目 -> `ProjectService.create_project()` -> `_activate_project()` -> `ProjectService.remember_project()` -> 写入 `config/settings.yaml`。
- 主工作台打开项目 -> `ProjectService.open_project()` -> `_activate_project()` -> `ProjectService.remember_project()` -> 写入 `config/settings.yaml`。
- 主工作台点击项目树 -> `ProjectService.open_project()` -> `_activate_project()` -> `ProjectService.remember_project()` -> 写入 `config/settings.yaml`。
