## 阶段 61：结果页 3D 视图默认隐藏坐标轴

当前阶段目标：
- 统一结果页所有 3D 可视化按钮的显示风格，默认隐藏 Matplotlib 3D 坐标轴和背景坐标盒。
- 本阶段只调整结果页独立 3D 视图的默认显示，不改 OpenFOAM 求解、不改数据读取、不改几何/CAD 页计算域预览。

本阶段完成功能：
- 结果页所有进入 `FoamDesk 3D 视图` 独立窗口的 3D 图默认隐藏坐标轴。
- 压力点云、压力表面、所选字段表面图默认不再显示 X/Y/Z 坐标轴。
- 速度箭头、速度切面、连续速度切面、速度流线默认不再显示 X/Y/Z 坐标轴。
- STL 预览、真实 OpenFOAM 3D Case、3D 技术验证场景也跟随同一默认行为。

本阶段页面与 UI 完成情况：

结果页 `ResultsTab`：
- 页面标题：`结果`，未调整。
- UI 组件：结果页按钮和下拉菜单未调整，仍通过 `VtkViewerDialog` 打开独立 3D 视图。
- 按钮或菜单项：所有原有 3D 按钮逻辑不变，只调整绘图默认样式。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：点击结果页 3D 相关按钮后，打开的图像默认隐藏坐标轴。
- 状态提示：本阶段未新增状态栏文案。

本阶段完成设计：
- 将坐标轴隐藏逻辑集中放到 `VtkViewerDialog._reset_axes()`。
- 所有结果页 3D 绘图函数都通过 `_reset_axes()` 创建 3D 坐标系，因此只改一个入口即可统一所有结果 3D 图。
- 保留图标题和色标，避免隐藏坐标轴后用户看不懂当前图对应的物理量。

本阶段数据流设计：
- 用户点击结果页 3D 按钮 -> 对应 `plot_*` 函数 -> `_reset_axes()` 创建 3D axes -> 默认 `axes.set_axis_off()` -> 绘制压力/速度/STL/流线数据 -> 显示独立窗口。
- 本阶段不读取或写入 JSON。
- 本阶段不修改 Case 文件和 OpenFOAM 输出文件。

本阶段核心业务场景完成进度：

- 项目选择业务场景：未变更。

- 未完成内容：如果后续需要，可增加“显示/隐藏坐标轴”开关，让用户临时查看坐标。
