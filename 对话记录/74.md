## 阶段 37：简单 STL 测试资产

当前阶段目标：
- 给 FoamDesk 增加一个内置的简单 STL 文件，方便用户不用自己找几何模型就能测试导入、snappy 配置、网格生成和流水线。
- 本阶段只提供测试几何资产，不改 UI、不改 OpenFOAM 执行逻辑。

本阶段完成功能：
- 新增 `assets/test_geometries/small_obstacle_cube.stl`。
- 该 STL 是一个封闭 ASCII STL 小立方体。
- 该立方体位于默认 `0..1` 计算域内部偏右上位置。
- 默认 `locationInMesh = (0.5, 0.5, 0.5)` 位于流体区域内，适合测试 snappyHexMesh。
- 新增 `assets/test_geometries/README.md`，说明该 STL 的用途和边界范围。

本阶段页面与 UI 完成情况：

几何/CAD 页 `GeometryTab`：
- 页面标题：未变更。
- UI 组件：未变更。
- 按钮或菜单项：未变更。
- 键盘操作：未变更。
- 鼠标操作：用户可以通过 `导入 STL` 按钮选择该测试文件。
- 状态提示：未变更。

本阶段完成设计：
- 测试 STL 放在 `assets/test_geometries`，与源码、运行时工作区分离。
- 文件采用 ASCII STL，方便查看和版本管理。
- 几何尺寸刻意避开 `(0.5, 0.5, 0.5)`，避免默认 `locationInMesh` 落入实体内部。

本阶段数据流设计：
- 用户点击 `导入 STL` -> 选择 `assets/test_geometries/small_obstacle_cube.stl`。
- 程序复制 STL 到当前 Case 的 `constant/triSurface`。
- 后续可点击 `生成 snappyHexMeshDict`、`一键前处理` 或 `一键仿真流水线`。
