## 阶段 21：用户自定义速度切面方向和位置

当前阶段目标：
- 让用户可以自己选择速度切面的方向和位置。
- 不再只能使用自动切面或固定中间切面。
- 本阶段继续使用点采样切面，不做连续插值面片和高级切平面算法。

本阶段完成功能：
- 结果页新增 `切面` 方向选择框。
- 切面方向支持：`自动`、`X`、`Y`、`Z`。
- 结果页新增 `位置` 数值输入框。
- 位置范围为 `0.00` 到 `1.00`。
- `0.00` 表示该方向最小坐标侧，`1.00` 表示该方向最大坐标侧，`0.50` 表示中间。
- 点击 `速度场 -> 加载速度切面` 时，会使用当前选择的方向和位置。
- 如果方向选择 `自动`，仍然会在 X/Y/Z 中选择速度变化最明显的切面。
- 结果提示中会显示请求方向、实际方向、归一化位置、真实切面坐标、点数和速度范围。

本阶段页面与 UI 完成情况：

结果页 `ResultsTab`：
- 页面标题：`结果`，未变化。
- UI 组件：`QComboBox`、`QDoubleSpinBox`、`QPushButton`、`QMenu`、`QTextEdit`、`FigureCanvas`。
- 按钮或菜单项：
- `速度场 -> 加载速度切面`：读取切面方向和位置后生成速度切面。
- `切面` 下拉框：可选择 `自动/X/Y/Z`。
- `位置` 输入框：可输入或点击调整 `0.00~1.00`。
- 键盘操作：可在位置输入框中输入数字。
- 鼠标操作：可点击下拉框选择方向，可点击数值框微调位置。
- 状态提示：结果区显示切面参数和速度范围；状态栏显示 `速度切面已加载。`

3D 视图 `VtkViewerDialog`：
- 页面标题：`Velocity Slice |U|`。
- UI 组件：Matplotlib 3D 坐标轴、切面采样点、颜色条。
- 按钮或菜单项：仍支持 `导出 PNG`。
- 鼠标操作：可旋转查看切面结果，可导出图片。
- 状态提示：没有切面点时显示 `No points near the selected slice.`。

本阶段完成设计：
- `VtkViewerDialog.plot_velocity_slice()` 扩展参数：`axis_name` 和 `normalized_position`。
- `axis_name=None` 或 `自动` 时，继续自动选择速度变化最明显的方向。
- 指定 `X/Y/Z` 时，按用户选择的方向生成切面。
- `normalized_position` 会被限制在 `0.0~1.0`，避免越界。
- `MainWindow._load_velocity_slice()` 读取 UI 控件值并传入绘图方法。
- 引入 `QDoubleSpinBox` 用于切面位置输入。

本阶段数据流设计：
- 用户设置 `切面=Y`、`位置=0.50`。
- 用户点击 `速度场 -> 加载速度切面`。
- `_load_velocity_slice()` 读取 `requested_axis` 和 `requested_position`。
- `OpenFoamVtkService.build_geometry_filter()` 读取当前 Case 和时间步。
- VTK 输出 `poly_data` -> 读取 `U`。
- `plot_velocity_slice(output, U, requested_axis, requested_position)`。
- 根据方向和位置筛选切面附近点。
- 计算 `|U|` 并绘制点采样切面云图。
- 结果区显示请求方向、实际方向、切面坐标、速度范围。
