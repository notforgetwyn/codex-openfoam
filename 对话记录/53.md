## 阶段 19：速度流线 v1

当前阶段目标：
- 在结果页增加速度流线图，让用户能看到流体大致沿什么路径流动。
- 本阶段先做固定种子点和稳定显示，不做用户自定义种子点、流线密度、流线长度和 VTK StreamTracer 高阶追踪。
- 继续使用 Python 客户端内部可视化，不调用 ParaView。

本阶段完成功能：
- 结果页新增 `加载速度流线` 按钮。
- 点击后读取当前 Case 的 `U` 速度字段。
- 自动判断主流向轴，例如 `X`。
- 自动在入口侧选择种子点。
- 根据真实速度场追踪并绘制多条流线。
- 流线颜色表示平均速度大小 `|U|`。
- 支持当前时间步选择器，例如 `0.5`。
- 如果当前 Case 没有 `U` 字段，会提示错误并列出可用字段。

本阶段页面与 UI 完成情况：

结果页 `ResultsTab`：
- 页面标题：`结果`，继续使用 `QLabel` 渲染。
- UI 组件：`QPushButton`、`QComboBox`、`QSpinBox`、`QTextEdit`、`FigureCanvas`。
- 按钮或菜单项：
- `加载速度流线`：可点击，触发 `_load_velocity_streamlines()`，读取当前 Case 的速度字段并打开 3D 流线视图。
- `时间步` 下拉框：速度流线读取当前选择的时间步。
- `加载速度箭头`：保持可用，用于看局部速度方向。
- `加载速度切面`：保持可用，用于看截面速度大小。
- 键盘操作：本阶段未新增键盘快捷键。
- 鼠标操作：点击 `加载速度流线` 后弹出或刷新独立 3D 视图窗口。
- 状态提示：结果文本区显示主流向轴、时间步、流线数量、流线点数、速度范围；状态栏显示 `速度流线已加载。`

3D 视图 `VtkViewerDialog`：
- 页面标题：`Velocity Streamlines |U|`。
- UI 组件：Matplotlib 3D 坐标轴、流线、起点标记、颜色条。
- 按钮或菜单项：无。
- 鼠标操作：可使用 Matplotlib 默认窗口查看图像。
- 状态提示：没有速度流线数据时显示 `No velocity streamline data to display.`；速度全为 0 时显示 `Velocity field is zero everywhere.`。

本阶段完成设计：
- `VtkViewerDialog` 新增 `plot_velocity_streamlines()`。
- `MainWindow` 新增 `_load_velocity_streamlines()`。
- 流线追踪采用真实 OpenFOAM `U` 字段和 NumPy 最近邻积分。
- 自动根据平均速度方向判断主流向轴，并从入口侧选择种子点。
- 当前 v1 优先可运行、可解释、不卡顿；后续可以升级为 VTK StreamTracer、可调种子点和更平滑的流线。

本阶段数据流设计：
- 用户点击 `加载速度流线` -> `MainWindow._load_velocity_streamlines()`。
- `_load_velocity_streamlines()` -> 读取当前 Case 和当前时间步。
- `_load_velocity_streamlines()` -> `OpenFoamVtkService.inspect()` 获取可用字段。
- `_load_velocity_streamlines()` -> `OpenFoamVtkService.build_geometry_filter(project, time_value)` 读取真实 OpenFOAM 数据。
- VTK 输出 `poly_data` -> 读取点字段 `U`。
- `U` 字段 -> NumPy 计算速度大小和平均速度方向。
- `plot_velocity_streamlines()` -> 根据主流向选择入口侧种子点 -> 最近邻积分追踪路径 -> Matplotlib 绘制流线。
- 绘图完成 -> 结果页文本区更新流线说明 -> 状态栏更新。
