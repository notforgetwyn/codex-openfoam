## 阶段 26：3D 视图多标签页

当前阶段目标：
- 把 3D 独立窗口从“每次加载覆盖上一张图”升级为“同一个 3D 窗口里保留多个视图标签页”。
- 让用户可以同时打开压力云图、速度箭头、速度切面、速度流线等多个 3D 图，并像浏览器一样切换查看。
- 本阶段只改 3D 视图窗口的承载方式，不改 OpenFOAM 求解、不改 VTK 数据读取、不新增新的物理后处理算法。

本阶段完成功能：
- `FoamDesk 3D 视图` 独立窗口新增标签页容器。
- 每次加载一个 3D 图都会新增一个标签页，而不是覆盖旧图。
- 支持同时保留多个图，例如压力表面图、速度箭头图、速度流线图。
- 标签页标题来自当前图名，例如 `Pressure Point Cloud`、`Velocity Vectors |U|`、`Velocity Streamlines |U|`。
- 如果重复打开同名图，标题会自动追加编号，例如 `3D Preview 2`。
- 新增 `清空视图` 按钮，可以一次清除当前 3D 窗口里的全部标签页。
- 标签页支持关闭，用户可以关闭不需要的 3D 图。
- `导出 PNG` 改为导出当前选中的标签页，而不是固定导出最后一次生成的图。

本阶段页面与 UI 完成情况：

3D 视图窗口 `VtkViewerDialog`：
- 页面标题：`FoamDesk 3D 视图`，保持独立窗口。
- UI 组件：使用 `QDialog` 作为独立窗口，使用 `QTabWidget` 管理多个 `FigureCanvas`。
- 按钮或菜单项：`清空视图` 可点击，触发后删除所有 3D 标签页。
- 按钮或菜单项：`导出 PNG` 可点击，导出当前选中的 3D 标签页。
- 标签页操作：每个新 3D 图生成一个标签页，用户可以点击标签切换。
- 标签页操作：标签页右侧关闭按钮可关闭单个视图。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：点击结果页的云图、速度场、3D 视图功能后，会在同一个 3D 窗口追加新标签页。
- 状态提示：主窗口原有加载完成提示保持不变。

本阶段完成设计：
- 将 `VtkViewerDialog` 从单一 `FigureCanvas` 改为 `QTabWidget + 多个 FigureCanvas`。
- `_reset_axes()` 不再清空同一张 figure，而是每次创建新的 `Figure` 和 `FigureCanvas`。
- `_tab_title()` 负责生成不冲突的标签页标题。
- `_export_png()` 从当前标签页读取 figure，保证导出的是用户当前正在看的图。
- `_clear_tabs()` 和 `_close_tab()` 负责释放不需要的标签页，避免 3D 图越开越多难以管理。

本阶段数据流设计：
- 用户点击 `云图 / 速度场 / 3D 视图` 中任意加载按钮 -> 主窗口读取当前 Case 和字段数据 -> 调用 `VtkViewerDialog` 绘图方法。
- `VtkViewerDialog._reset_axes(title)` -> 创建新的 `Figure` -> 创建新的 `FigureCanvas` -> 添加到 `QTabWidget` -> 设置为当前标签页。
- 用户切换标签页 -> `QTabWidget` 切换当前 `FigureCanvas` -> 用户看到对应 3D 图。
- 用户点击 `导出 PNG` -> 读取当前标签页的 `FigureCanvas.figure` -> 保存当前图像。
- 本阶段不新增 JSON 读写，不修改 OpenFOAM Case 文件。
