## 阶段 20：可视化图像导出 v1

当前阶段目标：
- 给 FoamDesk 的 3D 可视化窗口增加 PNG 导出能力。
- 本阶段先导出当前 Matplotlib 画布，不做批量导出、报告生成和导出参数配置。
- 支持当前已经实现的压力云图、速度箭头、速度切面、速度流线等图像。

本阶段完成功能：
- `FoamDesk 3D 视图` 窗口顶部新增 `导出 PNG` 按钮。
- 用户打开任意 3D 图像后，可以点击 `导出 PNG`。
- 弹出保存文件对话框，用户可以选择保存路径和文件名。
- 如果用户没有写 `.png` 后缀，程序会自动补上。
- 导出成功后弹出 `导出完成` 提示。
- 导出失败时弹出 `导出失败` 提示，显示文件写入错误原因。
- 默认文件名会根据当前图像标题生成，例如 `velocity_streamlines_u.png`。

本阶段页面与 UI 完成情况：

3D 视图 `VtkViewerDialog`：
- 页面标题：`FoamDesk 3D 视图`。
- UI 组件：`QPushButton`、`FigureCanvas`、`QFileDialog`、`QMessageBox`。
- 按钮或菜单项：
- `导出 PNG`：可点击，触发 `_export_png()`，把当前图像保存为 PNG。
- 图像区域：继续显示 Matplotlib 3D 图，包括云图、箭头、切面、流线。
- 键盘操作：本阶段未新增键盘快捷键。
- 鼠标操作：点击 `导出 PNG` 后选择保存路径；取消保存不会产生文件。
- 状态提示：导出成功弹出保存路径；导出失败弹出错误提示。

结果页 `ResultsTab`：
- 页面标题：`结果`，未调整布局。
- UI 组件：未新增结果页按钮。
- 按钮或菜单项：原有 `加载压力表面云图`、`加载速度箭头`、`加载速度切面`、`加载速度流线` 保持可用。
- 状态提示：结果页仍负责显示当前可视化说明，导出提示放在 3D 视图窗口内。

本阶段完成设计：
- `VtkViewerDialog` 新增 `_export_png()`。
- `VtkViewerDialog` 新增 `_last_plot_title`，每次 `_reset_axes(title)` 时记录当前图像标题。
- 默认导出文件名由当前图像标题生成，避免所有导出都叫同一个名字。
- 导出逻辑放在 3D 视图窗口内部，而不是结果页内部，因为导出的对象就是当前 3D 画布。
- 当前使用 Matplotlib `figure.savefig()` 导出，后续可以扩展为 SVG、PDF、批量导出和报告模板。

本阶段数据流设计：
- 用户点击任意可视化按钮 -> `VtkViewerDialog._reset_axes(title)` -> 记录 `_last_plot_title`。
- 用户点击 `导出 PNG` -> `VtkViewerDialog._export_png()`。
- `_export_png()` -> 根据 `_last_plot_title` 生成默认文件名。
- `_export_png()` -> `QFileDialog.getSaveFileName()` 获取用户保存路径。
- `_export_png()` -> 自动补 `.png` 后缀。
- `_export_png()` -> `Figure.savefig()` 写入 PNG 文件。
- 写入成功 -> `QMessageBox.information()` 提示导出完成。
- 写入失败 -> `QMessageBox.warning()` 提示错误原因。
