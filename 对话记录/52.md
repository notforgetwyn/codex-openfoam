## 阶段 18：速度切面图 v1

当前阶段目标：
- 在结果页增加速度切面图，让用户能看到某个截面上的速度大小分布。
- 本阶段先做自动中间切面，不做用户自定义切面位置、切面方向和连续面片插值。
- 继续使用 Python 内置可视化路线：VTK 读取 OpenFOAM 结果，Matplotlib 在 FoamDesk 内显示。

本阶段完成功能：
- 结果页新增 `加载速度切面` 按钮。
- 点击后读取当前 Case 的 `U` 速度字段。
- 自动比较 `X/Y/Z` 三个中间截面，选择速度变化最明显的中间截面显示。
- 在独立 `FoamDesk 3D 视图` 窗口中显示切面附近点的速度大小 `|U|`。
- 颜色表示速度大小，右侧色条显示速度范围。
- 支持当前时间步选择器，例如 `0.5`。
- 如果当前 Case 没有 `U` 字段，会提示错误并列出可用字段。

本阶段页面与 UI 完成情况：

结果页 `ResultsTab`：
- 页面标题：`结果`，继续使用 `QLabel` 渲染。
- UI 组件：`QPushButton`、`QComboBox`、`QSpinBox`、`QTextEdit`、`FigureCanvas`。
- 按钮或菜单项：
- `加载速度切面`：可点击，触发 `_load_velocity_slice()`，读取当前 Case 的速度字段并打开 3D 切面视图。
- `时间步` 下拉框：速度切面读取当前选择的时间步。
- `加载速度箭头`：保持可用，用于显示速度方向。
- 键盘操作：本阶段未新增键盘快捷键。
- 鼠标操作：点击 `加载速度切面` 后弹出或刷新独立 3D 视图窗口。
- 状态提示：结果文本区显示切面方向、切面位置、时间步、切面点数、速度范围；状态栏显示 `速度切面已加载。`

3D 视图 `VtkViewerDialog`：
- 页面标题：`Velocity Slice |U|`。
- UI 组件：Matplotlib 3D 坐标轴、切面采样点、颜色条。
- 按钮或菜单项：无。
- 鼠标操作：可使用 Matplotlib 默认窗口查看图像。
- 状态提示：没有速度切面数据时显示 `No velocity slice data to display.`。

本阶段完成设计：
- `VtkViewerDialog` 新增 `plot_velocity_slice()`。
- `MainWindow` 新增 `_load_velocity_slice()`。
- 切面选择策略从“固定 X 中面”升级为“自动选择 X/Y/Z 三个中面里速度变化最明显的切面”。
- 这样可以避免某些算例中固定切面速度全为 0，导致用户看到一张没有信息量的图。
- 当前 v1 使用点采样切面，后续可以升级为 VTK 真正切平面、插值和连续面片渲染。

本阶段数据流设计：
- 用户点击 `加载速度切面` -> `MainWindow._load_velocity_slice()`。
- `_load_velocity_slice()` -> 读取当前 Case 和当前时间步。
- `_load_velocity_slice()` -> `OpenFoamVtkService.inspect()` 获取可用字段。
- `_load_velocity_slice()` -> `OpenFoamVtkService.build_geometry_filter(project, time_value)` 读取真实 OpenFOAM 数据。
- VTK 输出 `poly_data` -> 读取点字段 `U`。
- `U` 字段 -> NumPy 计算速度大小 `|U|`。
- `plot_velocity_slice()` -> 比较 X/Y/Z 三个中间截面的速度范围 -> 选择最佳切面 -> Matplotlib 绘制切面采样点云图。
- 绘图完成 -> 结果页文本区更新切面说明 -> 状态栏更新。
