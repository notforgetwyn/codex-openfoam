## 阶段 77：结果页字段选择与显示方式选择重建

当前阶段目标：
- 按“结果场选择 + 显示方式选择”重建结果页。
- 删除旧的 `3D 视图`、`云图`、`速度场`、`字段时间`、`切面设置`、`流线设置`、`动画控制` 散装按钮。
- 本阶段先接入常用显示方式，不一次性完成全部高级后处理算法。

本阶段完成功能：
- 新增 `结果场选择` 模块。
- 新增 `显示方式选择` 模块。
- 变量下拉支持 `U`、`mag(U)`、`p`、`T`、`k`、`epsilon`、`omega`、`nut`、`yPlus`、`wallShearStress`、`vorticity`、`Q`、`alpha.water`。
- 新增时间步选择、颜色条范围、单位显示、最大值/最小值显示。
- 新增 `刷新字段/时间步` 和 `加载显示`。
- 已接入 `Surface`、`Slice`、`Vector`、`Glyph`、`Streamline`。
- `Contour`、`Iso-surface`、`Volume rendering` 已进入界面结构，后续接真实 VTK 实现。

本阶段页面与 UI 完成情况：

结果页 `ResultsTab`：
- 页面标题：`结果`，使用 `QLabel` 渲染。
- UI 组件：`QComboBox`、`QDoubleSpinBox`、`QPushButton`、`QLabel`、`QTextEdit`、Matplotlib 残差画布、独立 `VtkViewerDialog`。
- 按钮或菜单项：`结果数据` 保留刷新结果索引、导出求解指标、绘制残差曲线、导出 Markdown 报告。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：选择变量、时间步、显示方式后点击 `加载显示`。
- 状态提示：字段刷新、加载成功和错误都会写入状态栏或弹窗。

本阶段完成设计：
- 结果页从旧按钮堆叠改为两个业务模块。
- `结果场选择` 负责物理量、时间步、颜色范围和单位/范围反馈。
- `显示方式选择` 负责选择 Surface、Slice、Vector、Streamline 等显示算法。
- 三维显示仍使用独立窗口，避免 WSL 下 VTK 原生控件覆盖主页面。

本阶段数据流设计：
- `刷新字段/时间步` -> `OpenFoamVtkService.inspect()` -> 更新字段、时间步、单位和范围。
- `加载显示` -> `OpenFoamVtkService.build_geometry_filter()` -> 根据显示方式调用 `VtkViewerDialog`。
- `Surface` -> `plot_field_surface()`。
- `Slice` -> `plot_velocity_slice_contour()`。
- `Vector/Glyph` -> `plot_velocity_vectors()`。
- `Streamline` -> `_build_vtk_streamlines()` -> `plot_velocity_streamlines()`。
