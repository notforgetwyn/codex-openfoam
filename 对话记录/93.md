## 阶段 75：物性编辑器迁移到求解器准备页

当前阶段目标：
- 按页面职责重新整理：`求解器准备` 页面负责 `0/U`、`0/p`、patch 边界和流体物性。
- `仿真参数` 页面只负责 `system/controlDict` 的时间控制参数。
- 支持用户选择空气、水、机油、自定义流体，并填写 `rho`、`nu`、`mu`。
- 本阶段不做完整材料库、不做可压缩流 `thermophysicalProperties`，不做温度相关物性。

本阶段完成功能：
- `求解器准备` 页新增 `流体物性文件` 区域。
- 支持流体类型：空气、水、机油、自定义流体。
- 选择空气、水、机油后自动填入 `rho`、`nu`、`mu`。
- 新增 `按 rho × nu 计算 mu` 按钮，用户可根据密度和运动粘度自动计算动力粘度。
- 点击 `补齐边界和物性` 时，会读取当前物性输入并写入 OpenFOAM case。
- `仿真参数` 页移除流体物性输入，只保留 `endTime`、`deltaT`、`writeInterval`。
- `仿真参数` 页保存按钮现在只写 `system/controlDict` 的时间控制内容。

本阶段页面与 UI 完成情况：

页面名 `MainWindow / 求解器准备`：
- 页面标题：`求解器准备`。
- UI 组件：`QComboBox`、`QLineEdit`、`QPushButton`、`QTableWidget`、`QTextEdit`。
- 按钮或菜单项：`刷新求解器准备状态` 刷新 patch 表；`补齐边界和物性` 写入 `0/U`、`0/p`、`physicalProperties`；`按 rho × nu 计算 mu` 自动计算动力粘度。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：选择流体类型会自动应用预设；点击计算按钮会更新 `mu`。
- 状态提示：应用预设或计算 `mu` 后会在求解器准备状态区域提示。

页面名 `MainWindow / 仿真参数`：
- 页面标题：`仿真参数`。
- UI 组件：`QLineEdit`、`QSpinBox`、`QPushButton`、`QTextEdit`。
- 按钮或菜单项：`加载当前项目参数` 加载当前 `controlDict`；`保存参数到 Case` 只更新 `controlDict`；`恢复默认参数` 恢复默认时间参数。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：点击保存按钮后写入 `system/controlDict`。
- 状态提示：保存后显示 `controlDict 已保存`。

本阶段完成设计：
- `SimulationParameters` 新增 `material_name` 和 `dynamic_viscosity`。
- `OpenFoamCaseParameterService` 新增 `MATERIAL_PRESETS` 和 `material_preset()`。
- `OpenFoamCaseParameterService.save()` 继续负责写 `nu/rho/mu`，UI 不直接拼物性文件。
- `MainWindow._save_control_dict_parameters()` 专门负责仿真参数页的 `controlDict` 保存。
- 页面职责调整为：参数页管时间控制，求解器准备页管物性和边界。

本阶段数据流设计：
- 用户在求解器准备页选择流体类型 -> `_apply_material_preset()` -> 填充 `rho/nu/mu`。
- 用户点击 `按 rho × nu 计算 mu` -> `_calculate_dynamic_viscosity()` -> 更新 `mu`。
- 用户点击 `补齐边界和物性` -> `_read_parameter_inputs()` -> `case_parameter_service.save()` -> 写入 `physicalProperties/transportProperties`。
- 用户在仿真参数页保存 -> `_save_control_dict_parameters()` -> 只更新 `system/controlDict` 的 `endTime/deltaT/writeInterval`。
