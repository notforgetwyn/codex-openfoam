## 阶段 31：checkMesh 网格质量检查入口

当前阶段目标：
- 在已经支持 `blockMesh`、`snappyHexMeshDict` 生成和 `snappyHexMesh` 执行的基础上，增加 OpenFOAM 网格质量检查入口。
- 本阶段只做 `checkMesh` 命令执行、实时日志和中文摘要，不做网格质量图形化、不做自动修复、不做参数优化。

本阶段完成功能：
- 几何/CAD 页新增 `运行 checkMesh` 按钮。
- 顶部 `几何/CAD` 菜单新增 `运行 checkMesh` 菜单项。
- 点击前会检查当前 Case 是否已经有 `constant/polyMesh` 网格目录。
- 如果当前 Case 没有网格，会提示用户先运行 `blockMesh` 或 `snappyHexMesh`。
- 点击后通过 OpenFOAM 环境执行 `checkMesh`。
- 底部日志面板实时显示 `checkMesh` stdout/stderr。
- `checkMesh` 成功结束后，会生成中文摘要，说明网格是否 `Mesh OK`。
- 中文摘要会尝试提取点数量、面数量、单元数量、边界 patch 数量、最大长宽比、最大非正交角、最大扭曲度。

本阶段页面与 UI 完成情况：

几何/CAD 页 `GeometryTab`：
- 页面标题：`几何/CAD`，继续使用 `QLabel` 渲染。
- UI 组件：继续使用 `QPushButton`、`QTextEdit`、顶部 `QMenu`、底部日志面板和问题/诊断面板。
- 按钮或菜单项：
  - `导入 STL`：导入几何到 `constant/triSurface`。
  - `刷新几何清单`：刷新 STL 和 snappy 配置状态。
  - `生成 snappyHexMeshDict`：生成 snappy 网格配置。
  - `运行 snappyHexMesh`：生成或更新网格。
  - `运行 checkMesh`：检查当前网格质量。
  - `预览已导入 STL`：打开独立 3D 视图预览 STL。
  - `STEP/IGES 支持说明`：说明复杂 CAD 格式后续扩展路径。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：点击 `运行 checkMesh` 后切到底部日志面板，并实时显示 OpenFOAM 输出。
- 状态提示：
  - 运行中：`checkMesh 运行中。`
  - 成功：`checkMesh 完成。`
  - 失败：`checkMesh失败，退出码 X。`

本阶段完成设计：
- `MainWindow._run_check_mesh()` 复用已有 `QProcess + bash -lc + source OpenFOAM bashrc` 执行模式。
- `_active_process_kind` 增加 `checkMesh` 任务类型，让结束回调能区分不同 OpenFOAM 命令。
- `MainWindow._format_check_mesh_summary()` 负责把 OpenFOAM 原生日志转换成中文摘要。
- `MainWindow._process_label()` 统一维护任务类型到中文/命令名的显示映射。
- 本阶段仍未拆独立命令执行服务，等 OpenFOAM 命令继续增多后再抽象。

本阶段数据流设计：
- 用户点击 `运行 checkMesh` -> 检查当前 Case -> 检查 OpenFOAM 环境 -> 检查 `constant/polyMesh`。
- UI 创建 `QProcess` -> 执行 `source OpenFOAM bashrc && cd case && checkMesh`。
- OpenFOAM stdout/stderr -> `_read_process_stdout()` / `_read_process_stderr()` -> 底部日志面板。
- 进程结束 -> `_on_process_finished()` -> `_format_check_mesh_summary()` -> 写入问题/诊断面板和最近诊断摘要。
