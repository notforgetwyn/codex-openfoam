# Sprint 11：三维可视化技术验证

当前阶段目标：
- 验证 FoamDesk 能否使用 Python 库在客户端内部嵌入三维视图。
- 不依赖 ParaView 外部软件。
- 本阶段只做最小技术验证，不做真实 OpenFOAM 云图、切面、流线。

本阶段完成功能：
- 检查本机 WSL Python 环境：
  - `vtk` 可用。
  - `pyvista` 可用。
  - `pyvistaqt` 不可用。
- 选择更稳妥方案：直接使用 `VTK Python + QVTKRenderWindowInteractor` 嵌入 PySide6。
- `pyproject.toml` 新增 `vtk` 依赖声明。
- 结果页新增 `加载 3D 技术验证场景` 按钮。
- 结果页新增 VTK 三维渲染区域。
- 点击按钮后显示：
  - 单位计算域 wireframe 盒子。
  - 三维坐标轴。
  - 支持 VTK 默认相机交互。

本阶段完成设计：
- 三维可视化第一版选择 `VTK Python`，不是 ParaView 外部软件。
- 不引入 `pyvistaqt`，避免当前环境缺失依赖导致开发阻塞。
- VTK 画布直接嵌入 PySide6 页面，为后续真实 OpenFOAM mesh/field 可视化打基础。
- 本阶段只验证渲染链路，不解析真实场文件。

本阶段数据流设计：
- 用户点击 `加载 3D 技术验证场景`。
- `MainWindow._load_3d_preview_scene()` 创建 VTK cube source。
- VTK mapper -> actor -> renderer。
- renderer -> `QVTKRenderWindowInteractor`。
- PySide6 结果页显示三维技术验证场景。
