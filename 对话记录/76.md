## 阶段 39：3D 过滤自动刷新与 STL 放置预览

当前阶段目标：
- 让 3D 独立窗口中的显示范围设置可以立即刷新当前视图，不再要求用户重新点击加载按钮。
- 让用户导入 STL 前能看到缩放和平移后的放置位置，避免导入后才发现物体不在计算区域内。
- 本阶段只做导入前预览和当前 3D 标签页刷新，不做复杂拖拽式几何编辑、不做精确 STL 距离过滤。

本阶段完成功能：
- 3D 独立窗口的 `显示范围`、`STL`、`附近半径` 控件变化后，会自动重绘当前 3D 标签页。
- 每个 3D 标签页会记录自己的绘图回调，切换到不同标签页后修改过滤条件，会刷新当前选中的那张图。
- 速度流线也接入显示范围过滤，支持只看全部、所有 STL 附近或单个 STL 附近的流线。
- 导入 STL 时新增实时放置预览。
- STL 预览中灰色线框表示当前 MVP 的 1x1x1 计算区域参考，蓝色面表示当前缩放和平移后的 STL。
- 用户修改缩放、平移 X、平移 Y、平移 Z 后，预览图会立即更新。

本阶段页面与 UI 完成情况：

3D 独立窗口 `VtkViewerDialog`：
- 页面标题：`FoamDesk 3D 视图`。
- UI 组件：`QComboBox`、`QDoubleSpinBox`、`QTabWidget`、`FigureCanvas`。
- 按钮或菜单项：
  - `显示范围`：选择全部显示、所有 STL 附近、单个 STL 附近。
  - `STL`：选择具体 STL 对象，仅在单个 STL 附近过滤时有实际意义。
  - `附近半径`：控制 STL 包围盒向外扩展多少距离。
  - `清空视图`：关闭所有 3D 标签页。
  - `导出 PNG`：导出当前 3D 标签页图片。
- 键盘操作：本阶段未新增快捷键。
- 鼠标操作：用户切换显示范围或修改半径后，当前 3D 标签页自动刷新。
- 状态提示：如果过滤后没有数据，会在图中显示“当前显示范围内没有点数据 / 表面面片 / 速度流线”等提示。

STL 导入弹窗：
- 页面标题：`导入 STL：位置和缩放`。
- UI 组件：`QDoubleSpinBox`、`FigureCanvas`、`QDialogButtonBox`。
- 按钮或菜单项：
  - `OK`：确认当前缩放和平移参数并导入 STL。
  - `Cancel`：取消导入。
- 鼠标操作：调整缩放和平移数值后，预览图立即更新。
- 状态提示：如果 STL 无法预览，会在预览区域显示错误说明。

本阶段完成设计：
- `VtkViewerDialog` 增加每个标签页的刷新回调表，避免过滤参数只影响下一次加载。
- 当前过滤刷新采用“关闭当前标签页 -> 用原始数据重新绘制”的方式，保证图像和过滤参数一致。
- STL 放置预览使用 Matplotlib 直接读取源 STL 并按当前参数做临时坐标变换，不提前写入 Case 文件。
- 预览和正式导入解耦：只有用户点击 OK 后，才会真正写入 `constant/triSurface`。

本阶段数据流设计：
- 用户加载 3D 图 -> `VtkViewerDialog` 保存当前图的绘图回调 -> 用户修改过滤控件 -> 自动关闭当前标签页 -> 调用原回调重绘当前图。
- 用户选择 STL 文件 -> 打开导入弹窗 -> VTK 读取源 STL 顶点和三角面 -> 应用临时缩放和平移 -> Matplotlib 预览 -> 用户确认 -> `GeometryImportService.import_stl()` 写入 Case。
- 本阶段不新增 JSON 配置；STL 变换仍沿用上一阶段的 manifest 记录。

- 速度场业务场景：速度流线接入 STL 附近范围过滤。
