## 阶段 32：snappyHexMesh 参数面板

当前阶段目标：
- 让用户不再只能使用固定的 `snappyHexMeshDict` 默认参数。
- 本阶段先做最关键的 MVP 参数：加密等级、`locationInMesh`、是否启用边界层、边界层厚度。
- 本阶段不做高级网格参数、不做参数自动推荐、不做网格失败自动修复。

本阶段完成功能：
- 几何/CAD 页新增 snappy 参数区域。
- 新增 `最小加密等级` 输入框。
- 新增 `最大加密等级` 输入框。
- 新增 `流体内部点 locationInMesh` 的 X/Y/Z 输入框。
- 新增 `启用边界层 addLayers` 复选框。
- 新增 `最终边界层厚度` 输入框。
- 点击 `生成 snappyHexMeshDict` 时，会读取这些 UI 参数并写入 `system/snappyHexMeshDict`。
- 同时把参数写入 `constant/triSurface/snappy_config.json`，方便后续追踪。
- 如果最大加密等级小于最小加密等级，程序会自动把最大值修正为最小值。

本阶段页面与 UI 完成情况：

几何/CAD 页 `GeometryTab`：
- 页面标题：`几何/CAD`，继续使用 `QLabel` 渲染。
- UI 组件：新增 `QFormLayout`、`QSpinBox`、`QDoubleSpinBox`、`QCheckBox`。
- 按钮或菜单项：
  - `导入 STL`：导入几何到 `constant/triSurface`。
  - `刷新几何清单`：刷新 STL、snappy 配置状态。
  - `生成 snappyHexMeshDict`：读取当前参数面板并生成配置。
  - `运行 snappyHexMesh`：执行网格生成。
  - `运行 checkMesh`：检查网格质量。
  - `预览已导入 STL`：打开独立 3D 视图预览 STL。
- 键盘操作：本阶段未新增快捷键，但输入框支持正常键盘输入数值。
- 鼠标操作：用户可以用鼠标调整加密等级、内部点坐标、边界层开关和厚度。
- 状态提示：生成成功后日志会显示当前 snappy 参数，例如 `level=(1 2), locationInMesh=(0.5 0.5 0.5), addLayers=False`。

本阶段完成设计：
- `GeometryImportService` 新增 `SnappyHexMeshSettings` 数据类，作为 UI 和服务层之间的参数 DTO。
- `GeometryImportService.generate_snappy_hex_mesh_dict()` 支持接收 `SnappyHexMeshSettings`。
- `MainWindow._read_snappy_settings()` 负责从 UI 控件读取参数并做基础边界修正。
- `GeometryImportService._snappy_hex_mesh_dict()` 不再硬编码 `level (1 2)`、`locationInMesh (0.5 0.5 0.5)` 和 `addLayers false`。
- 本阶段保持 UI 层只收集参数，服务层负责生成 OpenFOAM 字典。

本阶段数据流设计：
- 用户在几何/CAD 页填写 snappy 参数 -> 点击 `生成 snappyHexMeshDict`。
- `MainWindow._read_snappy_settings()` -> 生成 `SnappyHexMeshSettings`。
- `MainWindow._generate_snappy_hex_mesh_dict()` -> 调用 `GeometryImportService.generate_snappy_hex_mesh_dict(project, asset_name, settings)`。
- 服务层写入 `system/snappyHexMeshDict`。
- 服务层写入 `constant/triSurface/snappy_config.json`，记录本次参数。

本阶段核心业务场景完成进度：
- 主菜单业务场景：未变更。
- 项目选择业务场景：未变更。
- Case 管理业务场景：继续保持当前 Case 隔离，参数只影响当前 Case 的 snappy 字典。
- 几何/CAD 业务场景：完成 STL 导入、snappy 配置生成、snappy 参数编辑、snappy 执行入口、checkMesh 检查入口。
- 求解业务场景：前处理配置能力增强，但还没有自动串联“生成字典 -> 运行 snappy -> checkMesh -> 求解”。
- 结果业务场景：未变更。
- 未完成内容：高级 snappy 参数、边界 patch 映射、加密区域、特征边提取、参数持久化回填 UI、参数推荐留到后续阶段。
